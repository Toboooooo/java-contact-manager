# Contact Manager (Java)

A simple console-based Java application that allows users to add, view, and remove contacts.

## Features
Add new contacts  
Display all saved contacts  
Remove selected contacts  
Interactive menu system  
Beginner-friendly code structure

## Technologies Used
- Java 21
- IntelliJ IDEA
- ArrayList
- Scanner

## How to Run
1. Clone the repository  
2. Open the project in IntelliJ IDEA  
3. Run `Main.java`  

## Purpose
This project was created for practice, learning, and portfolio development.  
It demonstrates understanding of:
- input handling
- loops
- conditionals
- data storage
- basic project structure
